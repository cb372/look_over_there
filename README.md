# あっち向いてホイ！

a.k.a. "Look over there!"

My first ever Rust program, written at Rust 超入門勉強会

http://akasaka-rust.doorkeeper.jp/events/16489

# To build

```
$ cargo build
```

# To run

```
$ target/look_over_there
```

# Example input/output

```
$ target/look-over-there
up
Chose direction: up
Got input: up

Gotcha!

$ target/look-over-there
down
Chose direction: right
Got input: down

OK, you win!
```
